# duplicated_petition( type $id )

He detectado que algunos navegadores, en algunos casos, envían varias veces la misma petición del formulario. En consecuencia se crean varios modelos (asientos, albaranes, etc...) con los mismos datos, es decir, duplicados. Para solucionarlo añado al formulario un campo petition_id con una cadena de texto aleatoria. Al llamar a esta función se comprueba si esa cadena ya ha sido almacenada, de ser así devuelve TRUE, así no hay que gabar los datos, si no, se almacena el ID y se devuelve FALSE.
