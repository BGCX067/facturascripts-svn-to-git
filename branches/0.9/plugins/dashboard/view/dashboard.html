Personalización de páginas

Puedes activar y desactivar páginas desde el menú "Admin -> Páginas" y quedarte con aquellos elementos que realmente necesites.

También puedes hacer que sólo algunos usuarios puedan acceder a ciertas páginas y delimitar sus tareas.

Ver administración de páginas

Plugins para personalización

Puedes personalizar tu instalación de FacturaScripts mediante su sistema de plugins, gratuitos y/o desarrollados a medida para tus necesidades.

Ya sea con plugins sectoriales o modificaciones personalizadas, tu sistema de facturación se adaptará a lo que realmente necesitas.
